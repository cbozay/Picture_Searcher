<h1>What is the study about?
<h6>Whether values get into input(search bar) exist are controlled by pressing "enter button". If there is the value on the system, the related picture comes as an output on the screen. Otherwise, on the screen none is shown.

<h3>How it's made?
<h6>Firstly, visuallity and responsiveness have been supplied with css. And then, a logical relation has been set up between search bar and all pictures.
